## 🎮 用户使用流程

### 模板选择和切换
1. **初次选择**:
   ```
   创建房间 → 选择游戏 → 选择模板 → 确认选择
   ```

2. **重新切换**:
   ```
   房间内 → 当前游戏区域 → 切换游戏 → 选择新模板 → 确认切换
   ```

3. **视觉反馈**:
   - 当前选择的模板有蓝色边框
   - 右上角显示"当前选择"徽章
   - 悬停效果和过渡动画
